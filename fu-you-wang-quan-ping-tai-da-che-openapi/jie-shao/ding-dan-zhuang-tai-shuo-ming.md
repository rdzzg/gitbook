# 订单状态说明

## 订单状态返回

| 状态码 | 描述 |
| :--- | :--- |
| pending | 呼叫中 |
| waiting | 等待接驾 |
| arrived | 司机抵达 |
| processing | 行程中 |
| pending\_pay | 等待支付 |
| completed | 已完成 |
| cancel |  已取消（部分平台会因为无司机应答主动取消订单 |
| timeout | 超时（呼叫无司机应答） |
| binvalid |  无效（服务提供商的客服介入时，可能判定某些有争议订单无效\) |
